Om Nom Run

Game Description

Run alongside Om Nom in his famous adventure, now finally in immersive and full-responsive HTML5. Try to run as far as possible by avoiding the dangerous obstacles and enemies, collect all coins and transform Om Nom into the fastest runner possible. Use the power-ups as smart as possible to explore the beautiful world with all it's beautiful areas, but be aware of the dangers that lurk inside of them.

How to Play

Tetris rules are simple and straightforward:

  • Control seven different shaped blocks falling from the top of the screen
  • Use arrow keys to move and rotate blocks for optimal arrangement
  • When a complete row of blocks is formed, that row disappears and scores points
  • As the game progresses, blocks fall faster
  • The game ends when stacked blocks reach the top of the screen

Controls:

  • Left/Right arrow keys: Move block horizontally
  • Up arrow key: Rotate block
  • Down arrow key: Accelerate block falling
  • Space bar: Drop block instantly to the bottom (hard drop)
  • P key: Pause game

Game Features

  • Classic Tetris gaming experience
  • Seven different block shapes, each presenting unique arrangement challenges
  • Increasing game difficulty with level progression
  • Score system and achievement tracking
  • Simple to learn but difficult to master gameplay mechanics
  • Classic game music and sound effects

Tips & Strategies

  • Try to keep the playing field as flat as possible, avoiding "peaks" and "valleys"
  • Reserve space for the I-shaped block (long bar), as it's key for clearing multiple rows
  • Learn the rotation properties of each block to quickly determine optimal placement
  • Use the "hard drop" (space bar) function to place blocks faster, but ensure correct positioning
  • Plan multiple steps ahead, not just the currently falling block
  • At faster game speeds, prioritize stability over scoring
  • Practice advanced techniques like "T-spin" for advantages in high-level play
